GREYED is a premium marketplace experience for curated vape and cigarette releases.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Pages

- Home
- Catalog
- Product detail
- Cart
- Auth (sign in, sign up)
- Account
- Admin dashboard shell

## Notes

- Styling lives in `src/app/globals.css` and shared components in `src/components`.
- Product data is mocked in `src/lib/catalog.ts`.
